# Spectral Suppression — ℓ=2 Lamb Mode Tidal Coupling

Reproducible calculation supporting

> *Spectral Suppression of Center-of-Mass Coupling to Gravitational Tidal Fluctuations* (2026).

If gravity is quantized, geodesic separations experience stochastic tidal fluctuations that couple to the **mass quadrupole moment**. Nearly every current experiment monitors **center-of-mass (COM) translation**, whose lab-frame quadrupole oscillates at twice the trap frequency (~10⁵ rad/s). The **internal ℓ=2 quadrupolar elastic (Lamb) mode** of the same particle oscillates at ~10¹¹ rad/s. This code computes, from first principles, how much more strongly the internal mode couples to a tidal noise spectrum that rises with frequency.

This repository contains no fitted parameters and no Monte Carlo — it is a closed elasticity + quantum-mechanics calculation. Every number in the paper is regenerated by `run_analysis.py`, and an independent suite of physics/numerics checks is in `verify.py`.

## Key results (fused silica, R = 100 nm)

| Quantity | Symbol | Value |
|---|---|---|
| Reduced eigenfrequency | η = ω₀R/v_t | **2.628** |
| Lamb frequency | f₀ | **15.7 GHz** |
| Geometric coupling factor | α | **0.84** (order unity, computed) |
| Matrix-element ratio (ω_COM=10⁵) | \|Q_body\|²/\|Q_COM\|² | **≈ 90 (10²)** |
| Spectral suppression at n=5 | (ω₀/ω_COM)⁵ | **≈ 10³⁰** |
| Combined advantage at n=5 | Γ_internal/Γ_COM | **≈ 10³²** (10³²–10³³ over trap range) |
| COM rate scaling | Γ_COM | **∝ ω³** (recovers Toroš et al.) |
| Internal rate scaling | Γ_internal | **∝ ω⁴** |

## What it computes

1. The fundamental ℓ=2 spheroidal ("Lamb") eigenmode of a free, homogeneous, isotropic elastic sphere — radial functions from two scalar potentials, traction-free secular equation, eigenvector. Bessel derivatives are analytic (the spherical Bessel ODE supplies j″), so the boundary conditions vanish to machine precision.
2. The induced symmetric-trace-free **mass quadrupole** of the mode and the dimensionless **geometric factor α** (the quantity earlier work left as "order unity").
3. The **matrix-element ratio** between the one-phonon internal channel and the two-phonon COM channel.
4. The **spectral-density ratio** for a tidal noise spectrum S_EE ∝ ωⁿ (the correspondence principle forces n ≥ 1; graviton-bath, vacuum FDT, and dimensional arguments give n = 5).
5. The **combined advantage** — the ratio of decoherence/heating rates — and the parameter-free **rate scalings**.
6. The **material spread** (silica, silicon, sapphire, diamond).

## Usage

```bash
python run_analysis.py    # prints every number and table in the paper
python verify.py          # runs 10 independent checks; exits nonzero on failure
```

`lamb_mode.py` and `coupling.py` are importable:

```python
import lamb_mode as lm, coupling as cp
g = cp.geometric_factor(lm.SILICA, R=100e-9)
print(g["eta"], g["alpha_frob"])                       # 2.628, 0.837
print(cp.combined_advantage(lm.SILICA, 100e-9, omega_com=1e5, n=5))  # ~8.7e31
```

## Verification

`verify.py` confirms, independently of the main code path:

- η = 2.628 (fundamental ℓ=2 root);
- analytic U′, V′ agree with finite differences;
- both traction-free boundary conditions vanish at the eigenfrequency (~10⁻¹⁶);
- the eigenvector from σ_rr = 0 also satisfies σ_rθ = 0;
- the induced quadrupole is **trace-free** with Q_xx = Q_yy = −Q_zz/2 (angular identities b₁ = −a₁, b₂ = a₂);
- α = 0.84 and is order unity;
- the matrix ratio scales as ω_COM² and the combined advantage as ω_COM⁻³ (i.e. Γ_COM ∝ ω³, recovering Toroš et al.);
- the spectral suppression is ~10³⁰ at n = 5;
- the long-wavelength (uniform-field) coupling is valid (λ_grav/R ~ 10⁵).

## Assumptions and scope

- Free, homogeneous, **isotropic** elastic sphere; fundamental ℓ=2 mode. Crystalline targets are anisotropic — the material table uses isotropic-equivalent constants and is order-of-magnitude.
- Long-wavelength (uniform-field) multipole coupling, justified numerically.
- The calculation gives **relative** spectral weight and coupling. The **absolute** amplitude of gravitational tidal fluctuations is model-dependent and is not addressed here.
- Quantities are reported in two tensor conventions (Frobenius-invariant and zz-component); α ranges 0.68–0.84 accordingly.
