# Bounding Gravitational Tidal Noise — ℓ=2 Lamb-Mode Analysis

Reproducible calculation supporting

> *Bounding gravitational tidal noise with the internal elastic modes of levitated nanoparticles* (2026).

If gravity is quantized, geodesic separations experience stochastic tidal fluctuations that couple to the **mass quadrupole moment**. Nearly every current experiment monitors **center-of-mass (COM) translation** (~10⁵ rad/s); the internal **ℓ=2 quadrupolar elastic (Lamb) mode** of the same particle oscillates at ~10¹¹ rad/s, in the band where the tidal coupling concentrates. This repository computes, verifies, and archives the consequences: a first finite bound on the internal-band tidal noise, and a material-dependence discriminant that separates a tidal signature from intrinsic backgrounds.

No fitted parameters, no Monte Carlo — a closed elasticity + open-quantum-systems calculation. Every number is regenerated by `run_analysis.py` and independently checked by `verify.py`.

## Key results (fused silica, 100 nm, T = 100 mK, τ = 10⁵ s)

| Quantity | Value |
|---|---|
| Lamb frequency (η = 2.628) | **f₀ = 15.7 GHz** |
| Geometric coupling factor | **α = 0.84** |
| Channel ratio (internal/COM) | **~10²** (62 full-channel, 93 single-mode) |
| Combined advantage at n=5 | **~10³²** |
| **First finite bound** S_EE^min (Q=10⁵) | **2.3×10⁷ s⁻³** |
| Gap to linearized-gravity vacuum | **10³⁹** (bounds enhanced states only; never closes) |
| Material discriminant (tidal n=5, diamond/silica) | **163×**, backgrounds fall (Pearson −0.97) |
| Cutoff sign-inversion scale ω_c*(n=5) | **4.2×10¹⁰ rad/s** |
| Readout ε-threshold (classical C ≈ 1400; = quantum C at 100 mK) | **2.7%** |

## What it computes

1. **Elastic eigenmodes** — the fundamental ℓ=2 spheroidal Lamb mode of a free isotropic sphere (analytic Bessel derivatives → boundary conditions to machine precision).
2. **Tidal coupling** — the induced STF mass quadrupole, the geometric factor α, and the internal-vs-COM channel ratio (with the COM two-phonon factor D = 10 derived, and the mode-counting conventions made explicit).
3. **The first finite bound** — S_EE^min from a thermal-limited sensing floor, the linearized-gravity vacuum reference, the 10³⁹ gap, and the R⁻⁶ no-go showing the vacuum gap never closes.
4. **The material discriminant** — tidal heating (∝ ρ ω₀^{n−1}) vs the intrinsic backgrounds (thermal loss, gas), their near-orthogonality, and the sign-based discriminant that rests on n ≥ 1.
5. **The cutoff threshold** — the silica=diamond sign-inversion scale ω_c*(n), and the ε readout threshold.

## Usage

```bash
python run_analysis.py    # prints every number and table (runs in ~10 s)
python verify.py          # 23 independent checks; exits nonzero on failure
```

Modules are importable:

```python
import lamb_mode as lm, coupling as cp, sensitivity as sn
cp.geometric_factor(lm.SILICA, 100e-9)["alpha_frob"]   # 0.837
sn.bound(Q=1e5)                                         # 2.33e7  (S_EE^min, s^-3)
sn.cutoff_threshold(lm.SILICA, lm.DIAMOND, 100e-9, 5)   # 4.22e10 rad/s
```

## Verification

`verify.py` confirms, independently of the main code path: the ℓ=2 eigenvalue and machine-precision boundary conditions; analytic-vs-finite-difference derivatives; the trace-free quadrupole; α = 0.84; the COM two-phonon factor D = 10; the channel ratio (62/93); the bound (2.3×10⁷), its Q^−½ scaling, the vacuum reference and 10³⁹ gap; the no-go (gap ≫ 1 even at bar-detector scale); the material discriminant (tidal rises, backgrounds fall, Pearson −0.97); the cutoff threshold and Earth's c/r_s; and the ε-threshold.

## Assumptions and scope

- Free, homogeneous, **isotropic** elastic sphere; fundamental ℓ=2 mode. Isotropic-equivalent constants for crystals (frequencies ~10%).
- Long-wavelength (uniform-field) tidal coupling, justified numerically (λ_grav/R ~ 10⁵).
- The **bound** is model-independent; the vacuum reference and the discriminant *steepness* use the linearized-gravity index n = 5, but the discriminant *sign* rests on n ≥ 1 alone. Absolute detectability is not claimed.
